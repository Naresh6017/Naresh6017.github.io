Improving Web Service discovery by Hybridization of Ontology focussed concept clustering and interface semantics.

A new semantic aware web service discovery approach that helps to extract underlying data. Semantic clustering using NPMI, and Fuzzy C-means clustering replace the traditional K-Means clustering in the same environment. The proposed strategy yields an accuracy of 90.4%

Query Recommendation system to support exploratory search

Queries submitted in a search goal shift processes are extracted from search enginer logs using Machine Learning, and are used to build a search goal shift graph. Random walk algorithm is then used to obtain query recommendations. Word2Vec, semantic regularization have been incorporated.

Hypernymy based model to eradicate mid-air crisis incorporating IoT and Semantic Web

The model focusses on how sensory inputs coupled with stored data in the cloud, alongside the hypernymy detection concept of Semantic Web would substantially increase the accuracy of medical diagnosis in mid-air. This would in turn helps save more lives in times of crisis.
